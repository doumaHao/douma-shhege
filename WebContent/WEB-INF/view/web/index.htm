关于我们

上海贺格精密机械有限公司， 是一家专业制造塑胶焊接机系列产品的公司。公司在焊接方案设计，焊接工艺制订和设备制造技术上积累了多年经验。本公司设备涉及折叠滤芯成套生产线，汽车油滤红外线焊接，非标红外线焊接设备，超声波自动化焊接，汽车雷达冲焊一体机，汽车门板热铆机，汽车内饰件专用焊接机，非标自动化装备焊接，机器人自动焊接等  我公司地处中国经济最活跃的长江三角洲，工厂位于上海市奉贤区，靠近杭州湾，东邻东海大桥，区位优势明显，交通便捷。工厂拥用自主研发设计和生产能力，是目前国内最具有发展潜力的塑胶焊接设备生产厂家。
